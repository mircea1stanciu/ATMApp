
**Why it matters:**
- Tokens = words generated by AI
- Lower tokens = lower cost
- 1000 tokens ≈ 750 words (about 1 page)

**If responses get cut off:**
```bash

```

### 2. **Smart Model Selection** (Already Optimal)

```bash

```

**Model Switching Strategy:**
- **gpt-4o-mini**: 95% of queries (general questions)
- **gpt-4o**: Complex code generation (when needed)
- **gpt-4**: Critical tasks only (rare)

### 3. **Temperature Control** (Already Set)


**What it means:**
- 0.0-0.3: More focused, shorter responses = **cheaper**
- 0.7: Balanced = **good middle ground** ✅
- 0.9-1.0: More creative, longer responses = **more expensive**

**For maximum savings:**
```bash

```

### 6. **Monitor Usage** (Set This Up)

**Create monitoring dashboard:**
```python
# Track in your database

```

**Weekly cost report:**
```sql
SELECT 
    DATE(created_at) as date,
    COUNT(*) as messages,
    SUM(tokens_used) as total_tokens,
    SUM(estimated_cost) as cost
FROM chat_sessions
WHERE created_at >= DATE_SUB(NOW(), INTERVAL 7 DAY)
GROUP BY DATE(created_at);
```

## 🚨 Cost Red Flags (Watch Out For)

### ❌ DON'T:

1. **Use GPT-4 by default**
   ```bash
   DEFAULT_MODEL=gpt-4  # ❌ 30x more expensive!
   ```

2. **Set MAX_TOKENS too high**
   ```bash
   MAX_TOKENS=4096  # ❌ 4x more expensive than 1000
   ```

3. **No spending limits**
   - Could get surprise $100+ bill
   - Set limits in OpenAI dashboard!

4. **No rate limiting**
   - User could spam thousands of requests
   - Always limit requests per minute

5. **Ignore monitoring**
   - Track usage weekly
   - Set up alerts

### ✅ DO:

1. **Use gpt-4o-mini for everything**
   ```bash
   DEFAULT_MODEL=gpt-4o-mini  # ✅ Cheapest
   ```

2. **Start with lower MAX_TOKENS**
   ```bash
   MAX_TOKENS=1000  # ✅ Increase only if needed
   ```

3. **Set hard spending limits**
   - $5-10/month for testing
   - Increase as needed

4. **Monitor usage daily**
   - Check OpenAI dashboard
   - Track in your app

5. **Cache common responses**
   - FAQ answers
   - Documentation queries

```python
@app.get("/api/admin/usage-stats")
async def get_usage_stats(current_user: User = Depends(get_super_admin_user)):
    """Get AI usage statistics"""
    from sqlalchemy import func
    
    stats = db.query(
        func.count(ChatSession.id).label('total_chats'),
        func.sum(ChatSession.tokens_used).label('total_tokens'),
        func.sum(ChatSession.estimated_cost).label('total_cost')
    ).first()
    
    return {
        "total_chats": stats.total_chats or 0,
        "total_tokens": stats.total_tokens or 0,
        "total_cost": f"${stats.total_cost or 0:.2f}",
        "avg_tokens_per_chat": stats.total_tokens / stats.total_chats if stats.total_chats else 0
    }
```
