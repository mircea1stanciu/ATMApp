## 💡 Quick Commands

```bash
# Backend Development
cd backend
python -m venv venv
source venv/bin/activate
pip install fastapi uvicorn sqlalchemy langchain langchain-openai
uvicorn main:app --reload

# Frontend Development
cd frontend
npx create-next-app@latest . --typescript --tailwind --app
npm run dev

# Docker (All-in-One)
docker-compose up -d
```
